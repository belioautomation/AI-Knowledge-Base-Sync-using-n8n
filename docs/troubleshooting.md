# Troubleshooting

## Google Drive Trigger doesn't fire

- Check folder permissions.
- Verify OAuth credential.

---

## AI Agent returns invalid JSON

- Verify prompt.
- Enable Structured Output Parser.

---

## Notion cannot find database

- Connect the Integration.
- Refresh database list.

---

## Telegram not sending

- Verify Bot Token.
- Verify Chat ID.

---

## PDF extraction fails

- Ensure uploaded file is a valid PDF.
