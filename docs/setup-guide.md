# Setup Guide

## Requirements

- n8n
- OpenRouter API Key
- Google Drive
- Notion
- Telegram Bot

---

## Configure Credentials

### Google Drive

Connect OAuth2 credential.

### OpenRouter

Add API Key.

### Notion

Create an Integration.

Generate an Access Token.

Connect the Integration to your database.

### Telegram

Create a bot using BotFather.

Configure Bot Token.
